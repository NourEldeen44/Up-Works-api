For Web Sockets Documentation Please Follow These Instructions:

On Client Side For Realtime Chat a User Can Send Multible Socket Events using Socket.io

Sendable Events:

socket.emit("user connected", user_id);

socket.emit("join", chatRoom_id, user_id);

socket.emit("message", "message text");

User Can also Listen for Events

Listenable Events:

socket.on("new message", (message) => { //do some action with message });

socket.on("error", (msg) => { //handle err });
